YNAB API contract:
```
POST https://api.youneedabudget.com/v1/budgets/{budgetId}/transactions
Authorization: Bearer {YNAB_PERSONAL_ACCESS_TOKEN}
Content-Type: application/json

Body:
{
  "transaction": {
    "account_id": string,       // Manuel's or Emily-Kate's account ID
    "date": "YYYY-MM-DD",       // today's date
    "amount": number,           // milliunits, NEGATIVE for outflow (e.g. -12990 for £12.99)
    "payee_name": "Amazon",
    "memo": "Manuel — AirPods case",
    "cleared": "cleared",
    "approved": false,
    "category_id": null         // uncategorized
  }
}

Response 201:
{ "data": { "transaction": { "id": "uuid", ... } } }
```

    Export signature:
    ```typescript
    export interface YnabTransactionParams {
      budgetId: string;
      accountId: string;
      amount: number;        // in pounds/dollars (e.g. 12.99) — NOT milliunits
      description: string;
      senderName: string;    // e.g. "Manuel" or "Emily-Kate"
    }

    export async function createYnabTransaction(
      params: YnabTransactionParams
    ): Promise<string>  // returns transaction ID
    ```

    Implementation details:
    - Convert amount to milliunits: `Math.round(params.amount * 1000) * -1`
    - Date: `new Date().toISOString().split('T')[0]` (YYYY-MM-DD, today)
    - Memo: `${params.senderName} — ${params.description}` (em dash, not hyphen)
    - POST to: `https://api.youneedabudget.com/v1/budgets/${params.budgetId}/transactions`
    - Headers: `{ 'Authorization': \`Bearer ${process.env.YNAB_PERSONAL_ACCESS_TOKEN}\`, 'Content-Type': 'application/json' }`
    - Body: `{ transaction: { account_id, date, amount (milliunits), payee_name: 'Amazon', memo, cleared: 'cleared', approved: false, category_id: null } }`
    - On non-2xx response: `throw new Error(\`YNAB API error: ${res.status}\`)`
    - On success: return `data.data.transaction.id`
