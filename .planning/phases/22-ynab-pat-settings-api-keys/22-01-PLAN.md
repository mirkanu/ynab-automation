    Replace the 501 stub with a real implementation:

    5. If YNAB returns non-2xx, return 502 JSON `{error: 'YNAB API error: {status}'}`.
    6. Parse response and return 200 JSON `{budgets: [{id, name}]}` — only `id` and `name` fields, filter out nothing (all budgets returned).
    7. Wrap in try/catch — catch returns 500 JSON `{error: String(err)}`.

    Do NOT import from `src/lib/ynab.ts` `getValidYnabToken` (it is not exported). Implement the fetch directly.
  </action>
