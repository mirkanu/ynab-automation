  _Validated: eBay and Amazon both confirmed in production (05-02 human verify)_

  _Validated: error notification path unchanged from Phase 4_

### Category Tagging (code complete, human verify outstanding)

- [x] **CAT-01**: App detects a category name typed on the first line of the forwarded email
  _Implemented: extractCategoryHint() — human verify outstanding_
- [x] **CAT-02**: App fetches YNAB category list and matches typed name (case-insensitive, partial)
  _Implemented: getCategories() + findCategory() — human verify outstanding_
- [x] **CAT-03**: If a matching category is found, the transaction is created with that category
  _Implemented: categoryId threaded to createYnabTransaction — human verify outstanding_
- [x] **CAT-04**: If no category is typed, or no match found, transaction is created uncategorized
  _Implemented: silent no-match, no notification — unit tested_
