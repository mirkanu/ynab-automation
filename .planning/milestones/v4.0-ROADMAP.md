# Milestone v4.0: Admin Backend UI

**Status:** SHIPPED 2026-03-28

## Overview

Password-protected admin UI for configuration, monitoring, and testing — making the app fully self-serviceable without touching environment variables or raw logs.

## Milestone Summary

**Key Decisions:**
- iron-session v8.0.4 for cookie auth (Edge-compatible, zero DB dependencies)
- Inline CSS styles throughout admin UI (matching SetupWizard patterns)
- DB-backed settings (no restart required for config changes)
- Railway API token stored permanently for seamless saves
- Test mode toggle: process emails without YNAB writes
- YNAB account/budget dropdowns fetched live from API
- Activity log stores full trace including account name, memo, category name

**Quick Tasks Completed During Milestone:**
- Test mode toggle (parse emails without YNAB writes)
- DB-backed settings (instant config changes, no Railway restart)
- Login password reveal button
- Test mode banner on all pages, smart replay behavior
- Activity log YNAB entry details (account name, memo, category)

**Technical Debt Incurred:**
- None significant
